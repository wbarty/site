Simple Asset Pricing with Geometric Brownian Motion

Posted on 2020-09-08

Probably one of the simplest simulations we can do in python for finance is simulating an asset price based on a Geometric Brownian Motion. We first start with a log-normal random walk which is a basic Stochastic Differential Equation (SDE) with many applications in finance, serving as the building block of various more complicated SDE’s.

Thus, we use the continuous counterpart, Ito’s Lemma (formula) which is essentially a Taylor Series Expansion but with certain rules that we apply because of the Brownian Motion/Weiner process component of our equation, these arise because of the quadratic variation of the process. It isn’t hugely important to know how the quadratic variation works however it will help you understand why the rules are applied.

Using the above equation we can directly import it into python and set the constants equal to some relevant values, finally we iterate over the total time period and plot an asset price modeled by GBM.

import numpy as np
import random
from matplotlib import pyplot as plt

def gbm(S,v,r,T):
    return S * np.exp((r - 0.5 * v**2) * T + v * np.sqrt(T) * random.gauss(0,1.0))
    # apply the GBM solution using a gaussian (normal) R.V. with mean 0 and standar deviation 1
# Inputs
S0 = 100 # underlying price
v = 0.10 # volatility
mu = 0.05 # long run average
dt = 1/365 # time step length
T = 3 # period end
n = int(T/dt) # number of time steps

# GBM Process
path=[]
S=S0 # starting price
for i in range(1,n+1): # iterate over T with n steps (1095)
    S_t = gbm(S,v,mu,dt) # use our function
    S = S_t
    path.append(S_t) # apply to S_t

plt.plot(path)
plt.show()

Since this uses a normally distributed random variable each time you run the code you will get a new asset price path, the user can also fiddle around with mu and sigma to see the impacts they have on the path.
